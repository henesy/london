# London Bot
London is a bridge bot with no warranty. 

London is derived from [Glenda](https://github.com/henesy/glenda)

### Installing

```
go install github.com/henesy/london
```

### Usage
```
cd $GOPATH/src/github.com/henesy/london
go build
./london -t 'Bot BOT_AUTH_TOKEN'
```
